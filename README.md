# Bram Laravel Project

Dit is mijn Laravel project gehost via Laravel Cloud.  
Het project is lokaal gebouwd met:

- PHP 8+
- Composer
- Laravel Framework
- Git/GitHub voor versiebeheer

## Installatie (lokaal)
composer install  
cp .env.example .env  
php artisan key:generate  
php artisan serve  

## Deploy
Dit project wordt automatisch gedeployed via Laravel Cloud.
